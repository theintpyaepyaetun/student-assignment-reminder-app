# Student Assignment Reminder App

Week 05 - AI Assisted UI Prototype  
1305216 Mobile Application Development

## Project Overview
This project is a frontend UI prototype built using Flutter.
It focuses on clean architecture, reusable components, and mock data.

## Screens
- Login Screen
- Home Screen (Assignment List)
- Add Assignment Screen
- Settings Screen

## Architecture
The project follows a component-based structure:

lib/
  screens/
  widgets/
  main.dart

## Features
- Reusable AssignmentCard widget
- Clean folder structure
- Mock data UI prototype
- Prepared for backend integration

## AI Usage
AI was used to generate layout structure and reusable components.
Manual refinement was done for file organization and architecture design.
